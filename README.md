# Proyecto de Plataforma para Control de Flujo de Remesas

Este proyecto se compone de:
- Un archivo 'app.py' que contiene el código de la aplicación.
- Una carpeta 'templates' que contiene los archivos HTML de la aplicación.
- Una carpeta 'static' que contiene los archivos CSS y JS de la aplicación.
- Un archivo 'requirements.txt' que contiene las dependencias del proyecto.
- Un archivo 'Procfile' que contiene la configuración para desplegar la aplicación en Heroku.
- Una carpeta 'venv' que contiene el entorno virtual de Python.
